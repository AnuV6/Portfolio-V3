# Preview Image Placeholder

## Instructions:

1. Take a full-page screenshot of your portfolio homepage
2. Recommended size: 1920x1080px or larger
3. Save as: `preview.png` or `preview.webp`
4. Place it in `public/img/` directory

This will be displayed at the top of your README.md file.

### Tips for a great preview:
- Use a screenshot tool like:
  - Windows: Snipping Tool, ShareX
  - Mac: Command + Shift + 5
  - Browser: Full Page Screen Capture extensions
- Capture the hero section and first few sections
- Ensure all interactive elements are visible
- Use high resolution for clarity on GitHub
